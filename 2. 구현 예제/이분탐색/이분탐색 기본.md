# 이분 탐색 기본

```java
// 이분 탐색
static int BinarySearch (int l, int r, int key){
    if (l < r){
        int m = (l + r) / 2;

        if (key == arr[m]){ // * 이때 arr은 탐색의 대상이 되는 배열
            return 1;
        } else if (key < arr[m]){
            return BinarySearch(l , m-1, key);
        } else {
            return BinarySearch(m+1 , r, key);
        }
    }
    return 0;
}
```
